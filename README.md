# Simple_NEC_IR_Transmitter_PIC12F1572
Quick and dirty implementation of some functions (Volume up, volume down, mute, on-off) of a Toshiba CT-90429 remote control on a PIC12F1572 microcontroller in C.

![Quick and dirty](/images/quick_and_dirty_breadboard_build.jpg)

For testing purposes a white LED hass been used instead of a proper IR LED. The used white LED seems to issue enough IR radiation for operation in a few centimeters distance from the receiver. Do not recommend, but better than nothing ...
